    The difference between Descriptive and Inferential Statistics is as follows:

    • Descriptive Statistics: This branch aims to describe a chunk of raw data using summary statistics, graphs, and tables. Descriptive statistics are useful because they allow you to understand a group of data much more quickly and easily compared to just staring at rows and rows of raw data values⁴. Descriptive statistics describe a sample. You simply take a group that you're interested in, record data about the group members, and then use summary statistics and graphs to present the group properties⁵.
    • Inferential Statistics: Inferential statistics allow scientists to take findings from a sample group and generalize them to a larger population⁶. Inferential statistics make inferences about a larger population⁵.


    • Population: In statistics, a population is the complete set of items from which a statistical sample is drawn for a study¹. It can be a group of individuals, a set of items, etc. It makes up the data pool for a study⁴.
    • Sample: A sample refers to a smaller, manageable version of a larger group. It is a subset containing the characteristics of a larger population⁵. Samples are used in statistical testing when population sizes are too large for the test to include all possible members or observations⁵.
    • Attribute: In data analysis or data mining, an attribute is a characteristic or feature that is measured for each observation (record) and can vary from one observation to another[^10^]. It might be measured in continuous values (e.g., time spent on a website), or in categorical values (e.g., red, yellow, green)[^10^].
    • Variable: In statistical research, a variable is defined as an attribute of an object of study¹³. Choosing which variables to measure is central to good experimental design¹³. A variable is any characteristic, number, or quantity that can be measured or counted¹⁶.
    • Level of Measurement: There are four levels of measurement: nominal, ordinal, interval, and ratio. The level of measurement can influence the choice of statistical methods used to analyze the data, as some methods may be more appropriate for certain types of data than others¹⁷.
    • Dataset: A dataset (or data set) is a collection of data. In the case of tabular data, a dataset corresponds to one or more database tables, where every column of a table represents a particular variable, and each row corresponds to a given record of the dataset in question²². The dataset lists values for each of the variables for each member of the dataset²².

    4. Probability Sampling: This involves random selection, allowing you to make strong statistical inferences about the whole group¹. There are several types of probability sampling methods:

    • Simple Random Sample: Every member of a population has an equal chance of being selected to be in the sample⁴⁵.
    • Stratified Random Sample: The population is first split into groups. The overall sample consists of some members from every group. The members from each group are chosen randomly².
    • Cluster Random Sample: The population is first split into groups. The overall sample consists of every member from some of the groups. The groups are selected at random².

    Non-Probability Sampling: This involves non-random selection based on convenience or other criteria, allowing you to easily collect data¹. There are several types of non-probability sampling methods:

    • Convenience Sample: The researcher chooses a sample that is readily available in some non-random way².
    • Voluntary Response Sample: The researcher puts out a request for members of a population to join the sample, and people decide whether or not to be in the sample².


    • Pre-experimental Research Design: This is the simplest form of experimental research design. It is often used to test the effects of a treatment on a group³.
    • True-experimental Research Design: This is the most accurate form of experimental research design as it relies on the statistical hypothesis to prove or disprove the hypothesis³.
    • Quasi-Experimental Design: This type of design does not randomly assign subjects to treatment and control groups, making it less reliable for establishing cause-and-effect relationships³.
    • Randomised Block Design: This design is used when the experimenter wishes to control the variation in an experiment that is related to differences in test units³.
    • Completely Randomised Design: In this design, all experimental units have an equal chance of receiving any treatment³.
    • Between-Subjects Designs: In a between-subjects design, the various experimental treatments are given to different groups of subjects¹.
    • Factorial Design: An experiment might have just one independent variable or it might have several. This section describes basic experimental designs and their advantages and disadvantages¹.

      • Each design has its own strengths and weaknesses, and the choice of which to use will depend on the specific circumstances of the research. using System;
using System.Drawing;
using System.Windows.Forms;

class DrawingForm : Form
{
    public DrawingForm()
    {
        // Set the title of the window
        Text = "Drawing Shapes";

        // Set the size of the window
        ClientSize = new Size(400, 400);
    }

    protected override void OnPaint(PaintEventArgs e)
    {
        base.OnPaint(e);
        Graphics g = e.Graphics;

        // Create a Pen for drawing shapes
        Pen pen = new Pen(Color.Black);

        // Draw a line
        g.DrawLine(pen, 50, 50, 200, 50);

        // Draw a point
        g.FillEllipse(Brushes.Red, 100, 100, 5, 5);

        // Draw a circle
        g.DrawEllipse(pen, 150, 150, 100, 100);

        // Draw a rectangle
        g.DrawRectangle(pen, 50, 200, 150, 100);

        // Dispose of the Pen
        pen.Dispose();
    }

    public static void Main()
    {
        Application.Run(new DrawingForm());
    }
}
